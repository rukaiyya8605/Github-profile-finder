# Github-profile-finder
**GitHub Profile Finder** is a modern and responsive web application that allows users to search and explore GitHub profiles instantly using the GitHub API.
It displays user details such as profile picture, bio, followers, following, repositories, and account statistics in a clean UI with smooth animations and interactive effects.

✨ Features:

* Real-time GitHub profile search
* Repository and user statistics preview
* Search suggestions dropdown
* Search history using Local Storage
* Responsive modern UI/UX
* Dark/Light mode support
* Animated background & smooth transitions
* Instant profile preview with API integration

🛠️ Built With:

* HTML5
* CSS3
* JavaScript
* GitHub REST API
